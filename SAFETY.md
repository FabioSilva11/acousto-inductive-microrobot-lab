# Segurança, escopo e uso responsável

Este repositório é **apenas educacional e computacional**. Ele não autoriza, ensina nem valida uso real em humanos, animais, células em laboratório ou qualquer ambiente biológico.

## O que este projeto permite

- estudar modelos físicos aproximados;
- visualizar movimento em fluido viscoso;
- testar algoritmos de controle e enxame em simulação;
- estimar ordens de grandeza;
- criar documentação inicial para aprendizado.

## O que este projeto não permite

- construir dispositivo terapêutico real;
- aplicar ultrassom, cavitação, aquecimento ou campo eletromagnético em tecido vivo;
- tratar câncer, trombose, infecção ou qualquer doença;
- usar os parâmetros como orientação clínica;
- usar o módulo de interação celular para provocar dano real.

## Sobre “destruição de células”

O código usa um **índice abstrato de interação celular**. Ele não é um protocolo de ablação, não substitui estudos biomédicos e não deve ser interpretado como dose, pressão, frequência ou potência segura/efetiva.

Para qualquer pesquisa real, seriam necessários: comitê de ética, supervisão médica/biomédica, normas de biossegurança, ensaios in vitro, modelos animais quando cabível, validação por imagem, controle térmico, rastreabilidade de materiais e aprovação regulatória.
